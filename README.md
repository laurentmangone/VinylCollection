# Vinyl collection

App Android pour gerer une collection de vinyles en local (Room), avec liste, recherche, creation, modification et suppression.

## Fonctionnalites
- Liste avec recherche par titre et artiste
- CRUD via bottom sheet
- Stockage local avec Room

## Demarrer
Prerequis : Android Studio + SDK Android.

```zsh
./gradlew :app:assembleDebug
```

## Tests rapides
```zsh
./gradlew :app:compileDebugKotlin
```
